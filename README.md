# Game Admin Dashboard

遊戲平台公版管理後台系統，實際部署並運行於線上遊戲平台。

## 功能模組

- **玩家管理** — 帳號查詢、點數管理、封號操作
- **GM 指令系統** — 即時下達遊戲伺服器指令
- **財務對帳** — 交易流水查詢與手動核帳
- **客服工具** — 客服事務記錄與處理介面

## 技術棧

- **前端**：JavaScript / SCSS
- **後端**：Node.js / Express.js
- **資料庫**：MongoDB

## 說明

此為遊戲平台之內部管理工具，基於保密協議部分商業邏輯已移除，僅保留架構展示用途。
